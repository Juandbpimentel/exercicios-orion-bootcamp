# Branches do Repositório

Este repositório utiliza as seguintes branches:

## Branch Principal
- **main** - Branch principal do repositório

## Branches Auxiliares
- **dev** - Branch de desenvolvimento
- **stage** - Branch de homologação/staging
- **production** - Branch de produção

## Estrutura de Branches

```
main (principal)
├── dev (desenvolvimento)
├── stage (homologação)
└── production (produção)
```

## Políticas de proteção
- `main`: Proteção contra push, é obrigatório fazer um PR para enviar as as alterações.

## Notas
Todas as branches auxiliares foram criadas a partir da branch `main`.
